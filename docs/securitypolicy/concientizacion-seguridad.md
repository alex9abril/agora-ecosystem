# Programa de Concientizacion en Seguridad

## 1. Objetivo
Generar una cultura de seguridad con acciones de bajo costo y alta efectividad.

## 2. Alcance
Todo el personal y contratistas que tengan acceso a sistemas de Agora.

## 3. Frecuencia
- Una sesion anual obligatoria.
- Comunicaciones cortas cada 6 meses.
- Capacitacion de induccion para nuevas incorporaciones.

## 4. Temas minimos
- Phishing y suplantacion.
- Manejo seguro de contrasenas y MFA.
- Clasificacion y manejo de datos.
- Reporte de incidentes.

## 5. Evidencia
Lista de asistencia, materiales compartidos y fechas.
