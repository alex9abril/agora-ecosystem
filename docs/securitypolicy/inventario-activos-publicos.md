# Inventario de Activos Publicos

## 1. Objetivo
Mantener un inventario actualizado de dominios, subdominios, IPs publicas y
servicios expuestos a Internet.

## 2. Campos minimos del inventario
- Dominio o subdominio
- IP publica
- Proveedor / cuenta
- Servicio expuesto
- Responsable interno
- Fecha de ultima revision

## 3. Frecuencia de actualizacion
Mensual o tras cambios en infraestructura.

## 4. Verificacion
- Validar que dominios activos correspondan a servicios vigentes.
- Retirar del inventario activos obsoletos.

## 5. Evidencia
Archivo de inventario y registros de revision mensual.
