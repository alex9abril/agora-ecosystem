# Inventario de Activos TI

## 1. Objetivo
Mantener un inventario central de activos TI para control y monitoreo.

## 2. Alcance
Servicios, aplicaciones, bases de datos, repositorios, dispositivos criticos y
herramientas SaaS utilizadas por Agora.

## 3. Campos minimos
- Activo
- Tipo
- Criticidad
- Responsable
- Proveedor
- Fecha de ultima revision

## 4. Frecuencia
Revision trimestral y actualizacion cuando existan cambios relevantes.

## 5. Evidencia
Inventario actualizado y registro de revisiones.
