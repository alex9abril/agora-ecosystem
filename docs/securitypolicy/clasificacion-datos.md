# Clasificacion y Manejo de Datos

## 1. Objetivo
Definir categorias de datos y controles minimos de manejo.

## 2. Clasificacion
- **Publico:** Informacion publicada sin restricciones.
- **Interno:** Informacion operativa no publica.
- **Confidencial:** Datos de clientes, pagos, credenciales y cualquier PII.

## 3. Controles minimos
- Acceso por rol (minimo privilegio).
- MFA en cuentas con acceso a datos confidenciales.
- Prohibicion de compartir datos confidenciales en canales no autorizados.
- Registro de accesos en proveedores cuando exista soporte.

## 4. Manejo y transferencia
- Uso de canales cifrados para transferir informacion confidencial.
- Evitar descargas locales innecesarias.
- Accesos temporales cuando se requiera soporte externo.

## 5. Almacenamiento
Solo en proveedores aprobados y con cifrado en reposo.

## 6. Evidencia
Inventario de datos y controles aplicados.
