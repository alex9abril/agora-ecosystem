# Gestion de Riesgos de Seguridad

## 1. Objetivo
Identificar, evaluar, priorizar y tratar riesgos de seguridad.

## 2. Metodologia
- **Identificacion:** activos, amenazas, vulnerabilidades y controles actuales.
- **Valoracion:** probabilidad (1-5) e impacto (1-5).
- **Prioridad:** riesgo = probabilidad x impacto.
- **Tratamiento:** mitigar, transferir, aceptar o evitar.

## 3. Registro de Riesgos
Campos minimos:
- ID
- Activo afectado
- Amenaza
- Vulnerabilidad
- Impacto (1-5)
- Probabilidad (1-5)
- Riesgo (score)
- Plan de accion y responsable
- Fecha objetivo
- Estado

## 4. Criterios de aceptacion
- Riesgos con score alto requieren plan de mitigacion.
- Riesgos aceptados deben incluir justificacion de negocio.
